TaskFlow - Task Management Application
A full-stack task management web application with real-time updates, user authentication, and complete CRUD operations.

🚀 Features
User Authentication & Authorization

JWT-based secure authentication
User registration and login
Protected routes
Task Management (CRUD Operations)

Create new tasks with title, description, priority, and due date
Read/View all tasks with filters
Update existing tasks
Delete tasks
Real-time Updates

Live task updates using Socket.io
Instant synchronization across multiple devices
Advanced Features

Filter tasks by status (Pending, In Progress, Completed)
Filter tasks by priority (Low, Medium, High)
Responsive design for all devices
Professional UI with clean design
🛠️ Tech Stack
Backend
Node.js - Runtime environment
Express.js - Web framework
MongoDB - Database
Mongoose - ODM for MongoDB
JWT - Authentication
Socket.io - Real-time communication
bcryptjs - Password hashing
Frontend
React.js - UI library
React Router DOM - Navigation
Axios - HTTP requests
Socket.io-client - Real-time client
React Hot Toast - Notifications
date-fns - Date formatting
📋 Prerequisites
Before you begin, ensure you have the following installed:

Node.js (v14 or higher)
MongoDB (local or cloud)
npm (comes with Node.js)
